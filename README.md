# Ore price checker

Determine the most profitable HighSec ore to mine in EVE online.  
Calculates the % profit to be gained by compressing your ore.  
Uses the [EVEMarketer API](https://api.evemarketer.com/ec/).

### Install Dependencies

`npm i`

### Run

`npm run start`
